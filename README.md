# web-component-demo

Lezione di lunedì 19 gennaio 2026:

Il prof. ci ha mostrato un esempio di Web Component. Prima di tutto, ha spiegato che i Web Component sono una tecnologia che permette di creare componenti riutilizzabili e incapsulati per il web. Questi componenti possono essere utilizzati in qualsiasi pagina web senza preoccuparsi di conflitti con altri stili o script.

Domani, il prof. userà Angular per creare un'applicazione che utilizza Web Component. Ha detto che Angular supporta i Web Component e che possiamo integrare facilmente i componenti personalizzati nelle nostre applicazioni Angular.
